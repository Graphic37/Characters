This pack contains a lot of textures, mainly because of the full 52 playing cards deck, but also the dice textures. These textures are all found in both the Textures folder, but also duplicated in each filetype's folder (fbx/fbx(unity)/gltf/obj) for referencing/linking of the files. Just copy paste all contents of a filetype into your game engine, but on how your game engine imports things, you might want to do some re-organising of these files.

There are also extra textures included in the Textures folder, like alternate card backs and a four-colour deck variant, you can replace the regular textures with these in engine.
